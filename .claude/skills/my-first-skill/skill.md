# My First Skill

This is a simple skill to demonstrate how Claude Code skills work.

## When to Use This Skill

Use this skill when you want to:
- Understand the skill structure
- See a basic skill template
- Learn skill creation patterns

## Instructions

When this skill is invoked:

1. Acknowledge that this skill is active
2. Explain what skills are and how they work
3. Show the skill file structure

## Skill Structure

Skills are stored in: `.claude/skills/your-skill-name/skill.md`

The file must have:
- YAML frontmatter with `name` and `description`
- Markdown content with instructions for Claude

## Example Skill Invocation

Users invoke skills by typing the skill name or describing what they want to do.
